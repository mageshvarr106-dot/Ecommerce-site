🛒 E-Commerce Website (React)

A modern and responsive e-commerce web application built using React. This project demonstrates core shopping functionalities including product browsing, cart management, and order tracking.

🔗 Live Demo: https://ecommerce-site-su9d.onrender.com

🚀 Features:

* Dynamic product listing
* Add to cart functionality
* Cart quantity updates
* Order tracking system
* REST API integration
* Responsive design

🛠 Tech Stack:

* React.js
* JavaScript (ES6+)
* CSS
* REST APIs

📌 Note:
This project focuses on frontend development. Backend APIs are integrated from external services.

📂 Setup:

1. Clone the repository
2. Run npm install
3. Run npm run dev

📈 Future Improvements:

* Authentication system
* Payment integration
* Admin dashboard
